# Salon Web

React + TypeScript + Vite app for Supasalon.

## Scripts

- `pnpm dev` starts the Vite dev server.
- `pnpm build` runs TypeScript build and Vite production build.
- `pnpm lint` runs Biome lint checks.
- `pnpm preview` serves the production build locally.

## Tooling

This app uses Biome for linting and formatting via the root `biome.json` configuration.
